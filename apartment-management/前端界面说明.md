# 公寓管理系统前端界面说明

## 一、界面概述

已为公寓管理系统创建了一个现代化、响应式的前端Web界面，采用纯HTML+CSS+JavaScript实现，无需额外框架依赖。

## 二、功能特性

### 2.1 核心功能
✅ **公寓列表展示** - 分页显示所有公寓信息
✅ **搜索过滤** - 支持按名称、城市、状态搜索
✅ **新增公寓** - 完整的表单验证和数据提交
✅ **编辑公寓** - 修改公寓信息
✅ **删除公寓** - 带确认提示的删除功能
✅ **状态切换** - 快速启用/停用公寓
✅ **分页导航** - 完整的分页功能

### 2.2 界面特点
- 🎨 **现代化设计** - 渐变色背景、圆角卡片、阴影效果
- 📱 **响应式布局** - 自适应不同屏幕尺寸
- ⚡ **流畅动画** - 平滑的过渡和动画效果
- 🔔 **友好提示** - Toast消息提示
- 🎯 **直观操作** - 清晰的图标和按钮

## 三、技术实现

### 3.1 技术栈
- **HTML5** - 语义化标签
- **CSS3** - Flexbox/Grid布局、动画、渐变
- **JavaScript (ES6+)** - Fetch API、Async/Await
- **Font Awesome** - 图标库

### 3.2 API集成
所有API调用都基于后端提供的RESTful接口：

| 功能 | 方法 | 接口路径 |
|------|------|---------|
| 分页查询 | GET | /api/apartment/list |
| 新增公寓 | POST | /api/apartment/add |
| 编辑公寓 | PUT | /api/apartment/update |
| 删除公寓 | DELETE | /api/apartment/delete/{id} |
| 更新状态 | PUT | /api/apartment/status |

## 五、使用说明

### 5.1 启动步骤

1. **启动后端服务**
   ```bash
   cd apartment-management
   mvn spring-boot:run
   ```

2. **访问前端页面**
   ```
   http://localhost:8080
   或
   http://localhost:8080/index.html
   ```

### 5.2 操作流程

#### 新增公寓
1. 点击右上角"新增公寓"按钮
2. 填写表单信息（带*为必填项）
3. 点击"保存"按钮

#### 编辑公寓
1. 在列表中找到目标公寓
2. 点击"编辑"按钮（蓝色铅笔图标）
3. 修改信息后点击"保存"

#### 删除公寓
1. 点击"删除"按钮（红色垃圾桶图标）
2. 确认删除操作

#### 切换状态
1. 点击"启用/停用"按钮（黄色暂停/绿色播放图标）
2. 状态立即切换

#### 搜索过滤
1. 在搜索区域输入条件
2. 点击"查询"按钮
3. 点击"重置"清空条件
